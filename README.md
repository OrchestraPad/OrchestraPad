# OrchestraPad

Dieses Programm ist ein PDF-Viewer für Musiker namens **OrchestraPad**.

## So startest du das Programm:

1. Doppelklick auf die Datei **START.bat**.
2. Das Programm installiert kurz alles Nötige und öffnet dann automatisch deinen Browser.

*Hinweis: Falls es nicht startet, stelle sicher, dass Python (von python.org) auf deinem Computer installiert ist.*

## Noten hinzufügen
Lege deine PDF-Dateien einfach in den Ordner `usb_drive`. Klicke in der Web-Oberfläche auf **Bibliothek aktualisieren**.
